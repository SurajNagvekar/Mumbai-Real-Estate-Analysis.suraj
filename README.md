# Mumbai Real Estate Analysis Dashboard by suraj nagvekar

## 📊 Overview
This project analyzes Mumbai housing data to uncover pricing trends, demand patterns, and geographic distribution using SQL and Power BI.

## 🔧 Tools Used
- SQL (Data Cleaning & Analysis)
- Power BI (Dashboard & Visualization)

## 📈 Key Insights
- Apartments dominate the market (>95% of listings)
- 2 BHK is the most common housing type
- South Mumbai and BKC are premium pricing zones
- Peripheral areas offer better affordability (lower ₹/sqft)

## 📍 Features
- KPI cards for pricing and area insights
- BHK distribution analysis
- Top expensive & affordable localities
- Geographic price distribution using map
- Property type segmentation
